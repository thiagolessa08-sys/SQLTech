Seção 03
Agentic Loop IA
1
Frontend envia contexto
Browser serializa messages[] com histórico + system prompt contendo contexto do banco e schemas SQL. Requisição via POST /api/chat.
messages[] system_prompt sql_schemas
2
Flask → Anthropic API
Flask envia requisição para Anthropic com tool query_database definida. Modelo decide se precisa consultar dados.
POST Anthropic API tool: query_database claude-haiku-4-5
3
Verifica stop_reason
Se stop_reason == "tool_use": extrai SQL gerado pelo modelo e executa via safe_sql() no SQLite.
stop_reason=tool_use safe_sql() municipal.db
4
Injeta tool_result e itera
Resultado SQL é injetado como tool_result no array de messages e nova requisição é enviada para o modelo. Máximo de 6 iterações.
tool_result messages.append() max_iter = 6
5
Retorna texto + marcadores CHART
Quando stop_reason == "end_turn", Flask retorna texto com blocos [CHART]{json}[/CHART] embutidos.
stop_reason=end_turn [CHART]{json}[/CHART]
6
Chart.js renderiza visualizações
Frontend parseia marcadores e renderiza gráficos interativos com Chart.js 4.
bar horizontalBar line area multiBar stackedBar doughnut pie
Loop iterativo · máx 6 ciclos · encerra em stop_reason = end_turn

Seção 06
Sistema de Seeds e Ciclo de Vida de Bases
init_seeds() · Startup
1
App inicializa e chama init_seeds() automaticamente no startup
2
Verifica se tabelas seeds já existem no SQLite para evitar reimportação
3
Lê arquivos de data/seeds/:
Lancamento.csv (~700k linhas)
Pagamento.csv (~544k linhas)
4
pandas lê CSV → DataFrame → df.to_sql() no SQLite com flag is_seed=1
5
Registra no _bases_catalog com metadata e timestamp de ingestão
Ciclo de Vida das Bases
base_lancamento / base_pagamento
Seeds oficiais · is_seed=1
PERMANENTE
base_{name} — upload usuário
Bases dinâmicas · is_seed=0
SESSÃO
Logout do usuário
DELETE WHERE is_seed=0
APAGA
Seeds após logout
Permanecem intactas — is_seed=1
PROTEGIDO
POLÍTICA DE ISOLAMENTO
Bases de upload são isoladas por sessão. O logout limpa automaticamente os dados temporários, preservando apenas seeds e dados fixos.
